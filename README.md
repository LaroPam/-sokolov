# Graves of the Last Light

Браузерный Vampire Survivors-like про выживание на проклятом кладбище. Работает целиком на чистом HTML5 Canvas без внешних зависимостей. Спрайты — открытые SVG (CC0) из каталога `assets/`.

## Запуск
1. Просто открой `index.html` в браузере — если он блокирует ES-модули по `file://`, автоматически подключится офлайн-бандл `offline.js`.
2. Альтернативно можно запустить локальный сервер (не обязательно, но удобно):
   ```bash
   python -m http.server 8000
   ```
   Затем открой `http://localhost:8000`.
3. На стартовом экране выбери одно из пяти видов оружия (иконки под названием игры) и нажми **Запуск** или Enter/Space, чтобы начать забег.

## Управление
- Движение: **WASD** или стрелки.
- Автоатака: срабатывает сама по ближайшей цели в радиусе автоатаки.
- При получении уровня выбирай одно из 3 улучшений, игра ставится на паузу до выбора.

## Особенности прототипа
- Четыре вида нечисти: медленные мертвецы, быстрые вампиры, толпы хлипких скелетов после пятой минуты и периодический босс Повелитель костей с полоской HP.
- Пять архетипов оружия с уникальными апгрейдами: меч (ближний размах), метательные клинки, арбалетные болты, залп стрел и магический жезл со взрывами.
- Символьные/иконные спрайты героя, врагов, снарядов и орбитальных духов, фон-плитка кладбища.
- Коллизии по тикам, взрывы, пронзание и орбитальные импульсы; боссовая шкала появляется во время схватки и скрывается после победы.
- Постоянный 60 FPS цикл с авто- и офлайн-бандлом, статистикой выживания и отображением HP/уровня/опыта.

## Ассеты
- Все файлы в `assets/` — оригинальные SVG, созданные для этого прототипа и выпущенные под CC0 1.0 (публичное достояние).
- Можно свободно заменять их своими картинками; пути объявлены в `src/assets.js` и используются как модульной сборкой, так и офлайн-бандлом.
