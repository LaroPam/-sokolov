# Echoes of the Last Web

Браузерный прототип в жанре Vampire Survivors-like про выживание в руинах цифрового интернета. Работает целиком на чистом HTML5 Canvas без внешних зависимостей. Спрайты — открытые SVG (CC0) из каталога `assets/`.

## Запуск
1. Просто открой `index.html` в браузере — если он блокирует ES-модули по `file://`, автоматически подключится офлайн-бандл `offline.js`.
2. Альтернативно можно запустить локальный сервер (не обязательно, но удобно):
   ```bash
   python -m http.server 8000
   ```
   Затем открой `http://localhost:8000`.
3. Дождись стартового экрана и нажми **Запуск** (или Enter/Space), чтобы начать забег.

## Управление
- Движение: **WASD** или стрелки.
- Автоатака: срабатывает сама по ближайшей цели в радиусе автоатаки.
- При получении уровня выбирай одно из 3 улучшений, игра ставится на паузу до выбора.

## Особенности прототипа
- Четыре вида врагов (Glitch Bug, Data Leech, Corrupted Crawler, Null Wraith) с растущей сложностью и шансом элиты.
- Символьные спрайты для героя, врагов, оружия и орбитальных дронов, а также плиточный фон в стиле цифрового постапокалипсиса.
- Снаряды с автонаведением и коллизиями по тикам, цепная молния и орбитальные импульсы в апгрейдах.
- Постоянный 60 FPS цикл с процедурным фоном, статистикой выживания и отображением HP/уровня/опыта.

## Ассеты
- Все файлы в `assets/` — оригинальные SVG, созданные для этого прототипа и выпущенные под CC0 1.0 (публичное достояние).
- Можно свободно заменять их своими картинками; пути объявлены в `src/assets.js` и используются как модульной сборкой, так и офлайн-бандлом.
