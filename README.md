# Echoes of the Last Web

Прототип браузерного roguelike в духе Vampire Survivors, работающий локально в браузере без сборки.

## Как запустить
1. Открой `index.html` в современном браузере **или** подними локальный сервер:
   ```bash
   python -m http.server 8000
   ```
   и перейди на `http://localhost:8000`.
2. Дождись загрузки стартового экрана.

## Как начать забег
- Нажми кнопку **«Запуск в руины»** или кликни по стартовому экрану.
- Альтернатива: нажми **Enter** или **Space**.
- Управление в игре: **WASD** или стрелки.
- Автоатака срабатывает автоматически по ближайшим врагам.

Если после клика/нажатия стартовый экран исчезает и появляется арена — игра запустилась корректно. Если этого не происходит, убедись, что файл открыт через локальный сервер (некоторые браузеры блокируют доступ к ES-модулям с файловой схемы).
